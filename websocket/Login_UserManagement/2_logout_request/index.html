    LOGOUT_REQUEST

    Command: LOGOUT_REQUEST

    Request message.
    {
    "COMMAND": "LOGOUT",
    "LOGIN_ID": "{{login_id}}",
    "CHANNEL": "{{channel}}",
    "LANG": "{{lang}}",
    "SESSION_KEY": "{{skey}}",
}
    Field Type Format in DB Req. Description
    Command String Yes "LOGOUT_REQUEST"
    LOGIN_ID String VARCHAR2(16 BYTE) Yes For User Mode: User Login ID
    For Client Mode: Client Login ID
    CHANNEL String VARCHAR2(4 BYTE) Yes For User Mode: 'WU'
    For Client Mode: 'WS'
    LANG String VARCHAR2(2 BYTE) Yes 'EN' - English
    'VN' - Vietnamese
    SESSION_KEY String VARCHAR2(32 BYTE) Yes Session key

    Logout success:
